📊 AI Job Market Dashboard

🔍 Project Overview

This project analyzes the current trends in the AI job market using interactive dashboards built in Power BI. It provides insights into job demand, required skills, salary trends, and location-based opportunities.

---

🎯 Objectives

- Analyze demand for AI-related roles
- Identify top skills required in the market
- Understand salary distribution
- Compare job opportunities across different locations

---

🛠️ Tools & Technologies

- Power BI
- Excel
- Data Cleaning & Transformation
- Data Visualization

---

📈 Key Insights

- High demand for roles like Data Analyst, Data Scientist, and AI Engineer
- Skills like SQL, Python, and Power BI are most in demand
- Metro cities have higher job opportunities
- Salary varies significantly based on experience and skillset
